V HTML máme dva druhy seznamů:

  • Bodové (unordered) seznamy - <ul>
  • Číselné (ordered) seznamy - <ol>

Položky seznamu se v obou případech určují pomocí <li>

<ul> <li>Sýr</li> <li>Máslo</li> <li>Chleba</li> </ul> <ol> <li>Sýr</li> <li>Máslo</li> <li>Chleba</li> </ol>

CSS vlastnosti seznamů umožňují:

  • Nastavit různé značky položek seznamu pro bodové seznamy
  • Nastavit různé značky položek seznamu pro řadové seznamy
  • Nastavit obrázek jako značku položky seznamu
  • Nastavit pozici značek seznamů

• Značky položek seznamu

Vlastnost list-style-type umožnuje změnit značení položek na velké množství různých symbolů.

Tady je pár možností:

ul { list-style-type: square; } ol { list-style-type: upper-roman; }

• Obrázek jako značka položky seznamu

Vlastnost list-style-image umožnuje nastavit značení položek jako obrázek.

ul { list-style-image: url('chleba.png'); }

• Pozicování značek položek v seznamu

Vlastnost list-style-position určuje pozici značení položek seznamu.

Tato vlastnost se zejména projevuje při psaní bodů v textu, kdy při použití inside je značení mezi textem a při použití outside mimo text.

Jako výchozí nastavení této vlastnosti je outside.

ul { list-style-position: inside; }

• Jednoduché nastavení v jedno řádku

Pro rychlejší použití a nastavovaní všeho dohromady, se může použít vlastnost list-style

ul { list-style: inside square url('chleba.png'); }
